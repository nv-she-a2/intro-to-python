The following commands are available:
1. list show => shows all the lists available
2. list use list_name => use a given list
3. list create list_name => create a list with given name
4. list delete list_name => delete a list with given name
5. todo add item_names => add a todo item to the selected list
6. todo all => show all the todo items in the chosen list
7. todo edit item_id new_item_name => edit the todo item providing the id and new title
8. todo remove item_id => remove a todo item
9. todo complete item_id => mark a todo item as completed
10. todo incomplete item_id => mark a todo item as incomplete
11. help => prints this...
12. quit => exit the application
